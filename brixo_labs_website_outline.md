## 2. Problem Section
**Layout:** Short paragraph, left-aligned. Max 2-3 sentences.

**Header:** The agent adoption gap

**Copy:**
Teams stall at every stage: wrong use case, missing data, no success metrics, no system to improve after deployment. The result is pilots that never reach production, or agents running with no accountability.

The problem isn't the technology. It's the process.

**Visual:** Funnel or pipeline graphic showing agents dropping off at each stage (use case > data > testing > metrics > production). Only a small fraction reaches the end. Reinforces the copy without repeating it.

## General Design Principles

- **Typography-driven.** No stock images. Let the copy and whitespace carry the page.
- **Two fonts max.** One serif or strong sans-serif for headers, one clean sans-serif for body.
- **Color palette:** Minimal. Dark/light contrast with one accent color for CTAs.
- **Page length:** Should scroll in under 60 seconds. If it feels long, cut more body copy.
- **No animations** beyond subtle scroll reveals. The content is the product.
